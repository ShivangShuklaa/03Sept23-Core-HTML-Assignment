HTML Tags

  1. <html>:

    The root element that wraps all the content on a web page.

  2. <head>:

    Contains metadata about the document, such as the title, character set, and links to external resources.

  3. <body>:

    Contains the main content of the web page, including text, images, links, and other elements.

  4. <title>:

    Sets the title of the web page, displayed in the browser's title bar or tab.

  5. <style>:

    Defines CSS rules directly within the HTML document.

  6. <script>:

    Embeds or references JavaScript code within the HTML document.

  7. <h1> - <h6>:

    Defines headings with varying levels of importance, where <h1> is the highest and <h6> is the lowest.

  8. <p>:

    Represents a paragraph of text.

  9. <a>:

    Creates hyperlinks to other web pages or resources.

  10. <img>:

    Embeds images on the web page.

  11. <ul>:

    Defines an unordered (bulleted) list.

  12. <ol>:

    Defines an ordered (numbered) list.

  13. <li>:

    Represents a list item within either <ul> (unordered) or <ol> (ordered) lists.

  14. <div>:

    A generic container for grouping and styling elements.

  15. <span>:

    A generic inline container often used for applying styles or scripting to a portion of text.

  16. <form>:

    Used for creating interactive input forms for user input and data submission.

  17. <input>:

    Defines various types of input fields within a <form>.
